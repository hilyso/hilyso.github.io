### 示例 1

拉取远程仓库到本地, 添加 123.txt后在推动至远程仓库

``` shell
git init 
git remote add origin git@github.com:hilyso/test.git
git pull origin master
echo 123>123.txt
git add 123.txt
git commit -m 'add 123.txt'
git push origin master
```
